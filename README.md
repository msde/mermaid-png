## Mermaid

Mermaid is great!  Unfortunately the live editor only exports to SVG.
Confluence sucks!  It doesn't support SVG.

## Installing

* yarn

## Converting a mermaid spec to a png

For spec "login.txt" run the following:
  * `yarn run png login.txt`
  * `ls login.txt.png`
